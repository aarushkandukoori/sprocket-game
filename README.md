# 🦊 Sprocket's Journey Home

A storytelling coding game that teaches kids (ages 8–12) the three building blocks of programming: **sequencing**, **loops**, and **conditionals**.

**▶️ Play it now: https://aarushkandukoori.github.io/sprocket-game/**

## The story
Sprocket, a little robot fox, is lost in an enchanted forest and needs to find her way back to her glowing den. You're the one who speaks robot — snap together command blocks, press **Run**, and watch Sprocket act out your code. She bonks into trees when the logic is off and sparkles when she grabs batteries along the way.

## What you'll learn
The forest teaches itself. Each chapter introduces exactly one new idea, because the puzzle demands it:

| Chapter | Story | Coding concept |
| --- | --- | --- |
| 1. The Long Way Home | A winding path home | **Sequencing** — put your steps in the right order |
| 2. The Winding Staircase | A repeating staircase | **Loops** — repeat a pattern instead of retyping it |
| 3. The Tangled Thicket | A twisting spiral | **Conditionals** — one "if" rule lets Sprocket decide for herself |

## How to play
1. Tap the command blocks to build Sprocket's code.
2. Use 🔁 **Repeat** loops and the 🌲❓ **If Tree → Turn** rule as you unlock them.
3. Press **▶ Run** and watch her go. Stuck? Tap 💡 for a hint.
4. Guide her to the 🏠 to finish a chapter and unlock the next one.

## Tech
A single self-contained `index.html` file — no frameworks, no build step, no dependencies. Just open it in any browser. It's built to be easy to extend: all the levels live in a `LEVELS` array near the bottom of the file, so you can add your own forest by editing a few lines.

## For grown-ups & educators
Failure is friendly here — bonking into trees is how the logic clicks, and hints are optional so kids try first. Works great for a classroom, a coding club, or a curious kid at home.
